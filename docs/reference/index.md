# Package index

<div class="section level2">

## All functions

</div>

<div class="section level2">

-   `build_vignette_path()` : create raw user content path for a
    vignette
-   `desc` : a list of 68 repository descriptions for packages in review
    in Nov 2025
-   `find_existing_branch()` : find branch name for an arbitrarily named
    submission to Bioconductor
-   `get_issue_repo_urls()` : given an issue stream, find the first URL
    mentioned
-   `get_repo_issues()` : given a github repo and token, enumerate all
    open issues
-   `get_repo_metadata()` : produce a list with repository metadata,
    README, and DESCRIPTION for a given repository
-   `list_github_dir()` : use GitHub API to list files in a repo folder
-   `vig2data()` : Use the extract\_data facility defined in ellmer's
    doc to obtain summary information about textual content. Originally
    tailored to vignettes in bioconductor; it is newly generalized to
    handle any pdf, html or text in URL.
-   `viglist_23_nov_2025` : URLs for vignettes of packages in review 22
    Nov 2025
